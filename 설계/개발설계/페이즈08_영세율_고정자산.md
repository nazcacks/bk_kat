## 1. 페이즈 개요

- **목표**: ① 부가세 영세율·수출 관련 부속서식(§25.14)과 기타 신고서류(§25.15)를 BK 부가세 모듈의 부속서식으로 생성하여 부가가치세신고서(SA-VAT-01)의 영세율 첨부·정합성 검증을 완성하고, ② 고정자산 마스터·감가상각 자동산출·결산 자동전표(§25.16, §11.5)를 구축하여 결산(P05)의 감가상각 반영과 세무(감가상각비조정명세서) 연계 기반을 확보한다.
- **범위 요약**: 영세율·수출서류 9종(1차) + 기타 신고서류 2종(1차) + 고정자산 7종(1차) + 환급·과세전환·기타 8종(2차) = 총 26메뉴.
- **선행조건**: P03 매입매출전표의 과세유형(12.영세/16.수출/64.매세) 전표 데이터가 영세율 부속서식 "불러오기"의 원천이며(§25.14 각 절), P01 거래처등록(금융거래처·환급창구운영사업자)·회사등록(사업장세무서)·업무용승용차등록(SA-BAS-10)이 마스터 원천이다(§25.15.5, §25.1.10). 고정자산은 P02 전기이월(고정자산·업무용승용차 초기이월 — §25.2)과 P05 결산자료입력 '감가상각반영'을 전후 연계로 갖는다(§25.16.7).
- **MVP 연관**: MVP 최우선(P03 전표·P01 COA)의 후속 신고·결산 페이즈. 부가세 신고(P06·07)와 동일 시기(2610~2611)에 착수한다(§22.2).

## 4. 화면별 상세설계

공통(전 화면): 진입경로 = 부가세 모듈(영세율·기타신고) / 고정자산 모듈 메뉴. 조회조건 공통 = 회계기간(기수)·신고기간(예정/확정)·사업장, RLS 테넌트 격리(§25.16.1, 00_개요 3절). 엑셀 변환·주민번호 별표 마스킹은 각 절 명시 화면에 적용.

### 4.1 영세율·수출관련서류 (1차)

#### 4.1.1 SA-VAT-29 수출실적명세서 (§25.14.1)
- **목적**: Master L/C 등 직접 수출(영세율) 사업자의 신고기간 단위 수출실적 제출 서식 생성.
- **입력 필드**:

| 필드 | 형식/규칙 | 필수 | 비고(근거 §25.14.1) |
|---|---|---|---|
| 수출신고번호 | 14/15자리 | Y | 번호 체계 검증 |
| 기타영세율건수 | 건수 | - | 수출신고번호 없는 기타영세율 |
| 선(기)적일자 | 일자 | Y | 환입 시 환입일 |
| 통화코드 | 코드 | Y | 원화수출은 KRW 환율 1, 엔화는 1JPY당 환율 |
| 환율 | 선적일 기준환율/재정환율 | Y | 소수점 규칙은 `TenantAccountingConfig`(M4) |
| 외화 | 수출신고서 (33)금액, 소수점 2자리 | Y | |
| 원화 | 외화×환율 또는 선적 전 환가금액 | Y(자동) | 불일치 시 마감오류 |
| 작성일자 | 일자 | Y | 기본 다음달 25일 |

- **기능**: ① 수출신고번호·통화·환율 환가 검증, ② 영세 전표(16.수출) 연동 불러오기, ③ 마감(전자신고 전제 — 원화금액=외화×환율 불일치 시 마감오류), ④ 엑셀 업로드.
- **검증**: 수출신고번호 자릿수(14/15), 환가 불일치 마감오류, 기수 기간 내 선적일자.
- **산출물**: 수출실적명세서. **연계**: 매입매출전표(16.수출), SA-VAT-01 영세율 조기환급·예정·확정 첨부, SA-VAT-02(MRI) 원화환가 적정성 검토. 외화 필드는 §11.16 환율 마스터(`FxRate`)·N5(통화·환율·외화금액·원화환산) 연계.

#### 4.1.2 SA-VAT-30 영세율첨부서류제출명세서 (§25.14.2)
- **목적**: 개별소비세 수출면세 서류 기제출/전산매체 제출 사업자의 관리용·신고용 명세 생성.
- **입력 필드**:

| 필드 | 형식/규칙 | 필수 | 비고 |
|---|---|---|---|
| 구분 | 1.명세서/2.목록/3.전체 | Y | 1.명세서 선택 시 마감 버튼 노출 |
| 서류명 | 텍스트 | Y | |
| 발급자 | 거래처등록 F2 | Y | P01 거래처 마스터 |
| 발급일자·선적일자 | 일자 | Y | |
| 통화코드·환율 | 코드·환율 | Y | |
| 당기 제출금액 | 외화·원화 자동 | Y(자동) | |
| 당기 신고 해당분 | 금액 | - | |
| 제출사유(신고용) | 1.개별소비세 신고서 함께/2.테이프·디스켓 | Y(신고용) | 전자신고 필수 |

- **기능**: 첨부서류 메타·제출사유 코드를 증빙으로 관리, 마감, 엑셀. **연계**: 거래처등록, SA-VAT-01, SA-VAT-07 전자신고.

#### 4.1.3 SA-VAT-31 내국신용장/구매확인서 전자발급 (§25.14.3)
- **목적**: 전자무역문서 내국신용장·구매확인서 공급재화·수출재화임가공용역(영세율) 부속서류 생성.
- **입력 필드**: 구분(기수·예정/확정), 입력단위(월별/기간별), 서류구분(1.내국신용장/2.구매확인서), 거래일자, 서류번호·발급일(직접입력), 거래처명·사업자등록번호, 금액.
- **기능**: ① 불러오기(12.영세 전표, 계정 401~420/901~930), ② 서류번호·발급일 메타 관리·영세 전표 집계, ③ **첨부서류 제출 위해 마감 필수**, ④ 정렬·엑셀.
- **검증**: 마감 미완료 시 첨부 제출 불가. **연계**: 매입매출전표(12.영세), SA-VAT-01, SA-VAT-43.

#### 4.1.4 SA-VAT-32 외화획득명세서 (§25.14.4)
- **목적**: 영세율 적용사업자 첨부서류(부가세법 24조·시행령 31~33조) 생성.
- **입력 필드**: 영세율 적용근거, 법정제출 서류명, 법정서식 제출가능여부(1.여/2.부)·일자·불능사유, 외화획득명세(공급일자·공급받는자·내용 — 직접입력 또는 12.영세/16.수출 전표 불러오기), 붙임서식(외환입금증명서·공급계약서·invoice·소포수령증), 과세유형·거래처 설정.
- **기능**: 영세율 근거·붙임서식 증빙 메타 관리, 영세 전표 연동 집계, 엑셀. **연계**: 매입매출전표(12/16 영세계열), SA-VAT-01, SA-VAT-43.

#### 4.1.5 SA-VAT-39 재화·용역 공급기록표 (§25.14.11)
- **목적**: 외교공관 등에 공급하는 재화·용역 부속서류 생성.
- **입력 필드**: 공급일자, 공급받는자(F2 코드도움 또는 직접입력), 품목·금액, 신고일자(기본 다음달 25일).
- **기능**: 외교공관 공급기록 메타를 증빙으로 관리. **연계**: SA-VAT-01, 매입매출전표(영세계열).

#### 4.1.6 SA-VAT-40 공급가액확정명세서 (§25.14.12)
- **목적**: 항공기 외국항행용역 영세율 사업자 부속서류 생성.
- **입력 필드**: 노선(노선등록 마스터, **전자신고 필수**), 수입금액(여객/화물/수화물/우편물/기타 5구분), 비고(통화코드 — 전자신고 필수·외화금액).
- **기능**: ① 노선·수입구분별 계정 매핑, ② 불러오기(12.영세/16.수출, 구분별 계정 401~420/901~930), ③ 영세 전표 집계. **연계**: 노선등록, 매입매출전표, SA-VAT-01, SA-VAT-43.

#### 4.1.7 SA-VAT-41 선박에 의한 운송용역 공급가액일람표 (§25.14.13)

#### 4.3.5 SA-FA-05 고정자산관리대장 (§25.16.5)

#### 4.3.7 SA-FA-07 월별 감가상각비 계상 (§25.16.7)
- **목적**: 결산시점별 감가상각비 계상 + 자동전표(결차/결대) 생성. 탭: 결산기준 감가상각계상/월별감가상각비명세/자산별 총괄표.
- **조회조건·기능**:

| 구분 | 내용 |
|---|---|
| 1) 결산기준 감가상각 계상 | ① 결산월(월별·분기별 — 월별은 1월부터 순차 조회, [결산자료입력] 12월 결산 시 1~12월 합계 반영; 전체양도·폐기 자산은 양도·폐기월 이후 상각 미계산), ② 경비구분(500~800번대, 전체 시 경비구분별 합계) |
| 2) 조회월삭제 | 선택 월 내역 삭제(**복구 불가**) 후 재조회 시 고정자산등록에서 재산출 |
| 3) 불러오기 | 고정자산등록 수정사항 반영(선택 결산월만) |
| 4) 월별감가상각비 명세 | 자산별 월별 상각비 현황 |
| 5) 자산별 총괄표 | 경비별·계정과목별 월별 상각비·합계 |
| 더보기 | ① 감가상각비 수정(누적상각비 수정, 수정 라인 파란색 — 불러오기 시 재계산 주의), ② 월할상각 설정(상각완료자산·폐기자산 월할상각 사용여부; 전체양도자산은 설정 무관 양도월까지 월할), ③ 음수상각비 찾기(월별 순차 미조회 시 음수 발생분 검색), 엑셀변환 |

- **BK 자동전표**: 결산 모듈의 월별 감가상각 계상 배치가 경비구분별 상각비를 **코스트센터 차원 자동분개(결차/결대)** 로 생성, 월할상각·음수보정 로직을 결산 계산엔진에 내장(§14 회사 배치 '월별 감가상각', §11.3 ①).
- **산출물**: 월별 상각비 명세·총괄표, 감가상각 자동전표(결산). **연계**: SA-FA-01, [결산자료입력] '감가상각반영', BF/JV 자동분개 파이프라인.

1. **영세율 정합 게이트**: 12.영세/16.수출 전표 → VAT-29·31·32·40~42 불러오기 → VAT-43 총합계가 SA-VAT-01 영세매출금액과 일치 → 마감 성공. 불일치 시나리오에서 마감오류 재현(§25.14.15).
2. **환가 검증**: VAT-29에서 원화≠외화×환율 데이터 마감 차단, KRW(환율 1)·JPY(1JPY당) 특례 통과(§25.14.1).
3. **고정자산 상각 정확성**: 정률/정액·월할(취득·양도월)·비망가액(5% vs 1,000원)·업무용승용차(정액 5년) 케이스별 당기상각범위액 검증(§25.16.1).
4. **자동전표 왕복**: FA-07 월별 계상 → 결차/결대 자동전표 → 결산자료입력 '감가상각반영' 반영 확인, 조회월삭제 후 재산출 일관성(§25.16.7).
5. **대사**: 결산 시 고정자산관리대장 ↔ 총계정원장 금액 일치, 자산변동(부분매각·폐기) 안분 상각누계 자동계산 검증(§25.16.1·5).
6. **전표 수집**: 차변 유·무형자산 분개 → 등록/미등록 판별 → 일괄등록, 중복 반영 없음(멱등)(§25.16.1).
7. **검증표 전수**: 8절 20개 항목 차단/경고 레벨 테스트, 전자신고 필수 필드 누락 케이스.
8. **권한·감사**: SOD(작성≠마감 승인), 마스킹·Step-up, `DataChangeLog` 전후값 기록 확인.
9. 2차 화면은 1차 공통 프레임(부속서식 헤더·불러오기·마감·엑셀) 재사용으로 회귀 없이 추가되는지 확인.
